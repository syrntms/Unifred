Unifred
====

## Overview
このアセットはUnityのエディター上でSpotlightやAlfredのような一時的なユーザー入力を受け付けるウィンドウを提供するアセットです  

## Usage
Window操作  
* 決定  
	* Enter  
* キャンセル  
	* Esc  
* 選択列の移動  
	* Up / Down / Tab / Shift + Tabキー  
* 複数選択  
	* Up / Down キーで移動の際にShift  
  
  
  
デフォルト機能  
* ゲームオブジェクトを選択中にcommand + y  
	* コンポーネントのメソッド呼び出し  
* command + g  
	* Hierarchy内検索  
* command + t  
	* Project内検索
